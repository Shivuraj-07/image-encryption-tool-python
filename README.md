# Image Encryption & Decryption Tool

This project is a simple Python-based image encryption and decryption tool using pixel manipulation techniques.

## Features

* Encrypt images using a secret key
* Decrypt encrypted images
* Pixel manipulation based encryption
* Simple and user-friendly implementation

## Technologies Used

* Python

## How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Run the program:

```bash
python image_encryptor.py
```

## Example

Input Image → Encrypt → Encrypted Image

Encrypted Image → Decrypt → Original Image
